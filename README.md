# 今天算数

一个极简 PWA，用来记录每天一个小约定、晚间结果和拖延原因。

## 本地运行

```bash
cd /Users/liugancheng/today-counts
python3 -m http.server 5177
```

然后打开：

```text
http://localhost:5177
```

## 当前功能

- 每天只记一个小约定，默认只需要输入一句话。
- 支持一天记录多面 flag，并用进度条记录推进比例。
- 完成标准、最小版本和提醒时间放在“更多设置”里，默认不用填。
- 晚上可以标记“完成了 / 没算数”，没完成时只点一个原因：目标太大、忘了、没开始、被打断。
- 查看月度认真日历、当天详情和一句复盘建议。
- 支持保存分享海报、发给朋友和复制分享文案。
- 支持浏览器本地通知。
- 支持添加到手机/桌面，作为 PWA 使用。

## 注意

当前版本的提醒依赖浏览器通知能力。页面关闭后是否稳定提醒，取决于浏览器和系统策略。要做稳定提醒，后续可以接服务端推送、微信提醒或短信。
