# 今天算数小程序

这是「今天算数」的微信小程序版本，路径为 `/Users/liugancheng/today-counts/miniprogram`。

## 当前状态

- 已做成原生微信小程序，不依赖 npm 和构建工具。
- 支持今日立 flag、记录进度、复盘原因、月历统计、保存分享海报、发给朋友、复制文案。
- 数据暂时保存在微信本地缓存里，适合先预览和内测。
- 订阅提醒入口已经预留，但正式推送需要在微信公众平台配置订阅消息模板，并接入后端或云函数定时发送。

## 本地预览

1. 安装并登录「微信开发者工具」。
2. 选择「导入项目」。
3. 项目目录选择：

   ```text
   /Users/liugancheng/today-counts/miniprogram
   ```

4. 还没有小程序 AppID 时，可以先使用测试号或保留 `project.config.json` 里的 `touristappid` 做本地预览。
5. 进入开发者工具后点击「编译」，再用「预览」生成二维码，在手机微信里扫码体验。

## 正式上线

1. 在微信公众平台注册或进入你的小程序账号，拿到小程序 AppID。
2. 把 `project.config.json` 里的 `appid` 从 `touristappid` 改成真实 AppID。
3. 在微信开发者工具里重新编译并预览。
4. 确认没有问题后点击「上传」，填写版本号和项目备注。
5. 到微信公众平台后台进入「版本管理」，把上传的版本提交审核。
6. 审核通过后点击发布。

## 订阅提醒

小程序不能像浏览器网页那样随意在后台本地弹通知。要做稳定的每日提醒，需要：

1. 在微信公众平台开通/申请订阅消息模板。
2. 把模板 ID 写入 `utils/config.js` 的 `reminderTemplateId`。
3. 在用户点击提醒按钮时请求授权。
4. 用云开发定时触发器或自己的服务端，在用户设定时间发送订阅消息。

当前版本只完成了前端授权入口和提醒时间设置，真正的定时发送建议作为下一步开发。
